# web-board
## https://react-web-board.web.app/

firebase, heroku에 호스팅하면 .git 폴더가 생성되어 git add 명령어가 불가능하다.
1. front, back 폴더의 .git을 삭제하고 레파지토리에 저장한다.
2. 수정 시 변경된 파일을 다시 호스팅 한다.
3. 1, 2를 반복한다.
4. 너무 스마트하지 않다. 해결방법이 무엇일까?
